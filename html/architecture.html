Core A

受控真相源原子能力

这些能力回答“能不能安全地找到和读取 truth material”。它们是 deterministic information service，不是 AI 不可用时才存在的 fallback。

  • resolve scope
  • find objects
  • read object card
  • read L2
  • dependency traversal
  • trace and grounding

Core B

托管 LLM agent 分析

这些能力回答“如何把 truth material 变成可用分析”。AI 可以规划、取证、表达 unknown 和 candidate，但不能拥有 truth authority。

  • /ask
  • PR diff impact
  • exception investigation
  • failed message analysis
  • test planning
  • governance review

Boundary Checks

什么能支撑事实，什么不能。

材料用途是否拥有 truth
L2 runtime object事实回答、诊断、测试规划 grounding可以支撑事实
dependency edge解释、影响候选、必要上下文加载可以支撑依赖事实
object card / L0 / L1导航、消歧、召回不能替代 L2
LLM draft分析草稿、表达、候选不能自行成为 truth
memory / trace feedback体验和检索辅助不能参与 truth validation
